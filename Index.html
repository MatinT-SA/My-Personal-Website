طراح و توسعه دهنده وب سایت - فارغ التحصیل کارشناسی مهندسی کامپیوتر

درود بر شما. محمد متین طاهرزاده شاه آبادی هستم. فارغ التحصیل کارشناسی مهندسی کامپیوتر (نرم افزار) از دانشگاه آزاد کرج.
در حوزه وب و طراحی و توسعه سایت فعالیت شخصی یا فریلنسری دارم.
سابقه تدریس زبان انگلیسی به مدت یک سال در موسسه زبان.
تسلط و تمرکز بر روی سی شارپ، .NET و React و همچنین WordPress.
در حال حاضر، پس از پایان خدمت سربازی، آماده دریافت پروژه یا همکاری با شما هستم.
در آینده قصد فراگیری زبان های برنامه نویسی Python و PHP را دارم.
علاقه مند به تکنولوژی و مطالب علمی. مسئولیت پذیر، پشتکار بالا و خلاق در زمینه طراحی سایت های جذاب.

نمونه کارها

  • بازی Pig Game که با VanillaJS نوشته و درش از استایل دهی و انمیشن های جذاب و حرفه ای استفاده شده.
  • یک پروژه MERN که در آن کاربر می تواند ایجاد اکانت کرده یا از اکانت گوگل استفاده کند و اتفاقات روزمره زندگی خود را در قالب پست به نمایش گذاشته، ویرایش کرده، حذف نموده و حتی سایر پست ها در صفحه مخصوص خود به همراه جزئیات ببیند و کامنت بگذارد. از قابلیت صفحه بندی و سرچ نیز در آن استفاده شده.
  • یک قالب مدرن responsive ( برگرفته از GPT-3). با استفاده از React. تجربه و رابط کاربری قوی. استفاده از سیستم flex و grid برای نمایش اجزا، به کارگیری CSS پیشرفته، پشتیبانی کردن از تقریبا تمامی دستگاه ها. استفاده از CSS-Gradient پیچیده و انیمیشن.
  • یک رابط کاربری Responsive که با استفاده از Material-UI کد زده شده. یک فروشگاه آنلاین با ReactJS که کاربر می تواند محصولاتی را به سبد خرید اضافه کرده، مشخصات خود را وارد کند و با پرداخت بانکی با استفاده از Stripe.js، خرید را نهایی کند.
  • یک رابط کاربری ساده Responsive با استفاده از کتابخانه React زده شده که برای یک سایت مسافرتی می باشد. تصویر پس زمینه با 3 حالت متفاوت تست شده و با استفاده از Router ها امکان جابجایی بین صفحات وجود دارد.
